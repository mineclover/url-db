# tag 속성 타입

## 개요
순서 없는 일반 태그. 중복 값 허용하지 않음.

## 검증 규칙
- 최대 길이: 50자
- 금지 문자: `,`, `;`, `|`, `\n`, `\t`
- 대소문자 구분하지 않음 (소문자로 변환)

## 제약사항
- 동일 URL에 같은 속성으로 중복 값 불가
- `order_index` 사용하지 않음

## 예시
```json
{
  "value": "programming",
  "order_index": null
}
```

## 에러 코드
- `validation_error`: 빈 값, 길이 초과, 금지 문자 포함
- `conflict`: 중복 값 존재
