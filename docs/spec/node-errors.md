# 노드 관리 에러 코드

## 에러 코드 매핑

### 노드 생성/수정
- `validation_error`: 필수 필드 누락
- `conflict`: 도메인 내 컨텐츠 중복

### 노드 조회
- `not_found`: 노드 존재하지 않음

### 노드 삭제
- `not_found`: 노드 존재하지 않음

## 검증 규칙
- `url`: 필수, 최대 2048자 (형식 검증 없음)
- `title`: 선택, 최대 255자 (비어있으면 자동 생성)
- `description`: 선택, 최대 1000자
- 컨텐츠와 domain_id는 수정 불가
