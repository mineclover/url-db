URL Database Server
Usage:
  url-db [options]

Options:
  -mcp-mode string
        MCP server mode: 'stdio' for stdin/stdout, 'sse' for HTTP server (default "sse")
  -db-path string
        Path to the database file (e.g., ~/url-db.db)
  -tool-name string
        Tool name for composite keys (default: url-db)
  -port string
        Port for HTTP server (default: 8080)
  -help
        Show help message
  -version
        Show version information

Examples:
  url-db                                          # Start HTTP server
  url-db -mcp-mode=stdio -db-path=~/url-db.db    # MCP stdio mode
  url-db -mcp-mode=stdio -db-path=~/work.db -tool-name=work
  url-db -help                                    # Show this help
